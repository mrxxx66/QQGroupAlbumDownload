# QQ群相册下载器用户指南

## 项目简介

QQ群相册下载器是一个基于Electron + Vue 3构建的桌面应用程序，旨在帮助用户批量下载QQ群中的相册图片，解决手动逐张下载效率低的问题。

## 功能特性

- 图形化界面，操作简单直观
- 支持选择特定QQ群和相册
- 批量下载相册中的所有图片
- 支持Windows便携版运行

## 系统要求

- Windows 7及以上版本
- 至少2GB RAM
- 至少500MB可用磁盘空间

## 快速开始

### 1. 安装应用

从GitHub releases页面下载最新版本的安装包，解压后运行可执行文件即可。

### 2. 输入群号

1. 在"输入群号"步骤中输入目标QQ群号
2. 点击"获取相册列表"按钮

### 3. 选择相册

1. 在"选择相册"步骤中，系统会列出该群的所有相册
2. 勾选需要下载的相册
3. 点击"开始下载"按钮

### 4. 开始下载

1. 在"处理任务"步骤中，系统会开始下载所选相册中的所有图片
2. 可以查看下载进度和状态

## 开发指南

如果您想参与项目开发，请参考以下内容：

### 环境搭建

1. 安装Node.js 16或更高版本
2. 克隆项目代码：
   ```
   git clone <repository-url>
   ```
3. 安装依赖：
   ```
   npm install
   ```

### 运行开发环境

- 启动Web开发环境：
  ```
  npm run serve
  ```
  
- 启动Electron开发环境：
  ```
  npm run electron-build-dev
  ```

### 构建生产版本

- 构建Web版本：
  ```
  npm run build
  ```
  
- 构建Electron桌面应用：
  ```
  npm run pack
  ```

## 问题排查

### 常见问题

1. **无法获取相册列表**
   - 检查群号是否正确
   - 检查网络连接
   - 确保该群存在公开相册

2. **下载速度慢**
   - 更换网络环境
   - 避开网络高峰时段

3. **部分图片下载失败**
   - 重试下载功能将在后续版本中实现

## 注意事项

- 请尊重他人隐私，仅下载自己有权访问的相册
- 不得用于商业用途或非法用途
- 遵守QQ群相关管理规定
